# Automatic Medicine Dispenser (Arduino)

## Overview
An Arduino-based automatic medicine dispenser that reminds users to take medication on time. The system rotates a pill carousel using a servo motor and alerts the user with an LED and buzzer until a confirmation button is pressed.

## Features
- Timed pill dispensing
- Servo-based carousel (4 compartments)
- LED and buzzer alerts
- Confirmation button to stop alerts
- 16×2 LCD status display

## Components
Arduino Uno, Servo Motor, 16×2 LCD, Push Button, LED, Buzzer, Breadboard.
